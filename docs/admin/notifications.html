🔔 Push Notifications

Send push notifications to all installed apps via Firebase Cloud Messaging.

🔧 Firebase Setup Required

To send push notifications, you need a Firebase project:

  1. Go to Firebase Console
  2. Create a project (or use existing)
  3. Add an Android app with your app's package name
  4. Download google-services.json and save it
  5. Go to Project Settings → Cloud Messaging
  6. Copy the Server Key (starts with "AAAA...")

Save these in Settings → Firebase Config
